\section{调试工具}
    在此次实验的扩展部分，%
    我们完成了基于串口的调试工具。

    通过PC端发送命令到实验板，%
    可以完成设置断点、启动、继续运行、%
    单步调试等多种功能。%
    并且可以在PC端查看寄存器的值，%
    查看各个模块间传递的信号。

    调试工具的使用方法与命令行下的GDB工具十分相似，%
    有GDB使用经验的人可以轻松上手。

    此外，调试工具在设计时就考虑到了复用的问题，%
    因此与原有VHDL代码之间的耦合度非常小，%
    只需要在原有代码上稍作修改即可运行。%
    修改部分也有代码自动生成工具予以辅助，%
    在PC端配置config文件后即可自动生成需要添加的代码。%

    调试工具的具体使用方式请参见调试工具使用手册。
