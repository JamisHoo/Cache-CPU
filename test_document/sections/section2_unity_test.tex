        \subsubsection{测试方法}
            通过拨码开关输入操作数、操作符检查运算结果正确性。%
            使用状态机，在高频下自动产生操作数、操作符，%
            并于VHDL内置运算结果比较，以检验在高频下连续运算结果的正确性。%
            除此之外，还需要检查高频下乘法速度是否满足需要。

        \subsubsection{测试用例}
            除了简单测例、边界测例外，还需要测试加减法运算的溢出。
            测例诸如%
            0xffffffff+1、%
            0xffffffff+2、%
            0x00000001-2等。

        \subsubsection{测试结果}
            所有运算操作结果正确，乘法运算可以在25MHz频率下一个周期内得出正确结果。
